# kvspace 最高标准设计

> 参考：etcd clientv3、FaRM/HERD RDMA KV、Go 标准库惯用法。
> 目标：易读、低延迟、接口设计正交完备。

---

## 一、设计原则

### 1.1 易读性

| 规则 | 说明 |
|------|------|
| **命名自文档** | `DelTree` 而非 `DelR` |
| **无隐式转换** | 接口不接受 `any`，调用方必须显式选择 `kvspace.Int(42)` 或 `kvspace.Str("ok")` |
| **零歧义错误** | 返回有语义的 sentinel error，不依赖 `err.Error()` 字符串匹配 |
| **Value 不 panic** | 所有访问器在类型错误时返回零值，不 panic；调试时 `fmt.Println(v)` 输出可读内容 |

### 1.2 低延迟

VM 热路径每条指令约 3-5 次 kvspace 操作（`Get PC`、`GetMany slots`、`Set PC/status`）。

| 实现 | 目标延迟 | 策略 |
|------|---------|------|
| Redis | 50-100 μs (loopback) | 连接池、pipeline |
| RDMA | **1-5 μs** | 单边 READ/WRITE、零拷贝、DMA 安全缓冲区 |

类型化 Value 消除 VM 热路径上的 `strconv` 往返：arithmetic/compare 结果以 typed Value 写入，读取时直接 `.Int()`/`.Bool()`，无重新解析。

### 1.3 接口设计

| 要求 | 说明 |
|------|------|
| **无 ctx 污染** | kvspace 是内存空间抽象，类比 `map[string]Value`，不在方法签名上传 ctx；超时由连接层配置（Redis pool timeout）或 Watch 的 `timeout` 参数控制 |
| **核心接口最小化** | `KVSpace` 只包含正交操作；扩展能力通过独立接口组合 |
| **批量操作类型化** | `SetMany([]KVPair)` 而非 `map[string]any`，保证顺序、类型安全 |
| **Watch/Notify 语义** | 监听单个 key 的值变化通知，不是通用消息队列 |
| **中间件可组合** | `Middleware` 函数类型允许无侵入地叠加 tracing/metrics/retry |

## 二、错误类型

```go
// internal/kvspace/errors.go

var (
    // ErrNotFound 在 Get 时 key 不存在，或 Watch 超时时返回。
    // Redis 实现映射 redis.Nil；RDMA 实现映射 NOT_FOUND。
    ErrNotFound = errors.New("kvspace: key not found")

    // ErrClosed 在连接已关闭后调用任意操作时返回。
    ErrClosed = errors.New("kvspace: connection closed")

    // ErrLinkLoop 在软链接解析超过最大跳数时返回。
    ErrLinkLoop = errors.New("kvspace: link loop detected")
)
```

**设计说明**：
- 只暴露调用方需要 `switch` 分支的错误。内部的截断数据、编码错误作为 `fmt.Errorf("kvspace: decode: %w", ...)` 包裹，不独立为 sentinel。
- 不定义 `ErrTypeMismatch`：类型错误属于调用方逻辑错误，由 `Kind()` 检查前置防御；accessor 返回零值而非 error（见 §三）。

## 三、Value 类型

```go
// internal/kvspace/value.go

// Value 是 kvspace 中存储的类型化值。
//   - 零值（IsNil()==true）表示"不存在"或"未初始化"。
//   - 一旦由构造函数创建，字段不可修改（逻辑不可变）。
//   - raw 字节由 Value 自身 owned，DecodeValue 在内部 copy，不与输入缓冲区共享。
type Value struct {
    kind string // vtype name（如 "int" "float" "bool" "str" "bytes" "tensor"）
    raw  []byte // 类型化原始字节
}

// ── 构造函数（kind 与 vtype.Name() 对齐）────────────────────────────────────

func Int(v int64) Value     { return Value{kind: "int", raw: encodeInt64LE(v)} }
func Float(v float64) Value { return Value{kind: "float", raw: encodeFloat64LE(v)} }
func Bool(v bool) Value     { b := byte(0); if v { b = 1 }; return Value{kind: "bool", raw: []byte{b}} }
func Str(v string) Value    { return Value{kind: "str", raw: []byte(v)} }
func Bytes(v []byte) Value  { c := make([]byte, len(v)); copy(c, v); return Value{kind: "bytes", raw: c} }

// Raw 构造任意 vtype 的 Value（用于第三方 vtype 扩展，如 "tensor"）。
// raw 会被复制，调用方可安全复用原缓冲区。
func Raw(kind string, raw []byte) Value {
    c := make([]byte, len(raw))
    copy(c, raw)
    return Value{kind: kind, raw: c}
}

// ── 判断 ─────────────────────────────────────────────────────────────────────

func (v Value) IsNil() bool  { return v.kind == "" }
func (v Value) Kind() string { return v.kind }

// ── 访问器（类型不匹配时返回零值，不 panic）────────────────────────────────

func (v Value) Int() int64 {
    if v.kind != "int" || len(v.raw) < 8 { return 0 }
    return decodeInt64LE(v.raw)
}
func (v Value) Float() float64 {
    if v.kind != "float" || len(v.raw) < 8 { return 0 }
    return decodeFloat64LE(v.raw)
}
func (v Value) Bool() bool {
    if v.kind != "bool" || len(v.raw) == 0 { return false }
    return v.raw[0] != 0
}
func (v Value) Str() string {
    if v.kind != "str" { return "" }
    return string(v.raw)
}
func (v Value) Bytes() []byte {
    if v.kind != "bytes" { return nil }
    return v.raw
}

// RawBytes 返回底层原始字节（任意 kind）。不拷贝，调用方不得修改。
func (v Value) RawBytes() []byte { return v.raw }

// ── Stringer（调试可读，不影响语义）────────────────────────────────────────

// String 实现 fmt.Stringer，输出格式为 "kind:repr"，例如：
//   int:42    float:3.14    bool:true    str:hello    nil    tensor:120B
// 注意：获取 str 类型的字符串内容请用 v.Str()，不要用 v.String()。
func (v Value) String() string {
    switch v.kind {
    case "":      return "nil"
    case "int":   return "int:" + strconv.FormatInt(v.Int(), 10)
    case "float": return "float:" + strconv.FormatFloat(v.Float(), 'f', -1, 64)
    case "bool":  return "bool:" + strconv.FormatBool(v.Bool())
    case "str":   return "str:" + v.Str()
    default:      return v.kind + ":" + strconv.Itoa(len(v.raw)) + "B"
    }
}

// ── 内建 kind 常量（与 vtype.Name() 对齐）───────────────────────────────────

const (
    KindInt    = "int"
    KindFloat  = "float"
    KindBool   = "bool"
    KindStr    = "str"
    KindBytes  = "bytes"
    KindTensor = "tensor"
)
```

**关键设计决策**：

| 决策 | 原因 |
|------|------|
| accessor 类型不匹配返回零值 | 调用方通过 `Kind()` 前置判断，无需处理 error；避免 panic 使 VM 可继续运行 |
| `Str()` 不叫 `String()` | `String()` 专供 `fmt.Stringer`，输出 `"str:hello"` 格式；`Str()` 返回纯字符串内容 |
| `raw` owned（DecodeValue 内部 copy） | 避免与 Redis 读缓冲区共享，防止静默数据损坏 |
| `Bytes()` 构造函数也 copy | 调用方传入的 `[]byte` 可能来自栈或短生命周期缓冲区 |

## 四、核心接口

```go
// internal/kvspace/kvspace.go

// KVPair 用于批量写入，顺序确定（非 map）。
type KVPair struct {
    Key string
    Val Value
}

// KVSpace 是 kvlang 的类型化 KV 存储抽象。
//
// 使用模式：
//   kv.Set("/vt/0/pc", kvspace.Str("init/[0,0]"))
//   v, err := kv.Get("/vt/0/pc"); pc := v.Str()
//   kv.Notify("/vt/0/status", kvspace.Str("running"))   // 通知值已变更
//   val, err := kv.Watch("/vt/0/status", 5*time.Second) // 阻塞直到收到 Notify
//
// Watch/Notify 语义：监听单个 key 的值变化通知，不是通用消息队列。
//   Notify(key, val) 向该 key 的等待者投递 val；不等价于 Set（不写持久值）。
//   Watch(key, timeout) 阻塞等待下一次 Notify；超时返回 (Value{}, ErrNotFound)。
//   若无 Watch 等待，投递缓冲至多一条，下次 Watch 立即返回。
//
// 软链接透明穿透：访问 linkpath/x 等价于访问 target/x。
// 链接本身不经 Value 编码，以原始 sentinel 形式存储（实现内部细节）。
type KVSpace interface {

    // ── 单点读写 ─────────────────────────────────────────────────────────
    // Get 返回 key 对应的 Value。key 不存在时返回 (Value{}, ErrNotFound)。
    Get(key string) (Value, error)
    // Set 写入 key-value 并维护目录索引。
    Set(key string, val Value) error
    // Del 精确删除一或多个 key，同时清理目录索引。
    Del(keys ...string) error

    // ── 批量读写 ─────────────────────────────────────────────────────────
    // GetMany 按序返回多 key 的 Value。key 不存在时对应位置返回 Value{}（不返回 error）。
    GetMany(keys []string) ([]Value, error)
    // SetMany 原子性地写入多个 key-value（实现尽力保证，非事务保证）。
    SetMany(pairs []KVPair) error

    // ── 目录操作 ─────────────────────────────────────────────────────────
    // List 返回 prefix 下的直接子项名（不含路径前缀）。
    List(prefix string) ([]string, error)
    // DelTree 递归删除 prefix 树。若 prefix 本身是软链接，仅删除链接。
    DelTree(prefix string) error

    // ── 变更通知 ─────────────────────────────────────────────────────────
    // Notify 向监听 key 的 Watch 调用者投递 val。
    // 不等价于 Set：val 是一次性通知信号，不写持久 KV。
    // 若当前无 Watch 等待，val 缓冲至多一条，下一次 Watch 立即返回。
    Notify(key string, val Value) error
    // Watch 阻塞等待 key 上的下一次 Notify，返回投递的 val。
    // 超时时返回 (Value{}, ErrNotFound)。
    Watch(key string, timeout time.Duration) (Value, error)

    // ── 软链接 ───────────────────────────────────────────────────────────
    // Link 创建软链接：访问 linkpath/x 透明地访问 target/x。
    Link(target, linkpath string) error
    // Unlink 删除 linkpath 处的链接本身（不影响 target 树）。
    Unlink(linkpath string) error

    // ── 生命周期 ─────────────────────────────────────────────────────────
    // DisConn 断开连接并释放资源。DisConn 之后调用任何方法返回 ErrClosed。
    DisConn() error
}
```

**接口方法变更对照**：

| 旧方法 | 新方法 | 改动原因 |
|--------|--------|---------|
| `Get(key) (string, error)` | `Get(key) (Value, error)` | 类型化 |
| `Gets(keys...) ([]string, error)` | `GetMany([]string) ([]Value, error)` | 批量不用 variadic；slice 参数更明确；类型化 |
| `Set(key, any) error` | `Set(key, Value) error` | 类型安全，禁止隐式 `fmt.Sprint` |
| `Sets(map[string]any) error` | `SetMany([]KVPair) error` | 有序 + 类型安全 |
| `DelR(prefix) error` | `DelTree(prefix) error` | 语义自文档（Recursive → Tree） |
| `Watch(key, timeout) (string, error)` | `Watch(key, timeout) (Value, error)` | 仅类型化返回值，签名结构不变 |
| `Notify(key, any) error` | `Notify(key, Value) error` | 类型安全 |
| `DisConn() error` | 不变 | — |

## 五、编码格式（TLV）

```
[1 byte kind_len][N bytes kind_name][4 bytes raw_len LE][M bytes raw_value]
```

| 字段 | 大小 | 说明 |
|------|------|------|
| `kind_len` | 1B | kind_name 的字节数（1~127，0 非法） |
| `kind_name` | N B | vtype name，`[a-zA-Z0-9_]` 字符集 |
| `raw_len` | 4B | raw_value 字节数，uint32 little-endian |
| `raw_value` | M B | 类型化原始数据 |

**完整编码示例**：

| kind | 编码（hex） | 说明 |
|------|------------|------|
| `int(42)` | `03 69 6E 74  08 00 00 00  2A 00 00 00 00 00 00 00` | kind_len=3, "int", raw_len=8, int64 LE |
| `str("hi")` | `03 73 74 72  02 00 00 00  68 69` | kind_len=3, "str", raw_len=2, UTF-8 |
| `bool(true)` | `04 62 6F 6F 6C  01 00 00 00  01` | kind_len=4, "bool", raw_len=1 |

**DecodeValue 必须 copy raw bytes**（owned 语义，防止 buffer 复用导致静默损坏）：

```go
// internal/kvspace/encode.go

func EncodeValue(v Value) []byte {
    if v.IsNil() { return nil }
    buf := make([]byte, 1+len(v.kind)+4+len(v.raw))
    buf[0] = byte(len(v.kind))
    copy(buf[1:], v.kind)
    binary.LittleEndian.PutUint32(buf[1+len(v.kind):], uint32(len(v.raw)))
    copy(buf[1+len(v.kind)+4:], v.raw)
    return buf
}

func DecodeValue(data []byte) Value {
    if len(data) == 0 { return Value{} }
    kindLen := int(data[0])
    if len(data) < 1+kindLen+4 { return fallbackStr(data) }
    kind := string(data[1 : 1+kindLen])
    if !isValidKind(kind) { return fallbackStr(data) }
    rawLen := binary.LittleEndian.Uint32(data[1+kindLen : 1+kindLen+4])
    start := 1 + kindLen + 4
    if len(data) < start+int(rawLen) { return fallbackStr(data) }
    // ★ 必须 copy：Value.raw 是 owned 字节，不能是输入 data 的切片
    raw := make([]byte, rawLen)
    copy(raw, data[start:start+int(rawLen)])
    return Value{kind: kind, raw: raw}
}

func fallbackStr(data []byte) Value {
    raw := make([]byte, len(data))
    copy(raw, data)
    return Value{kind: "str", raw: raw}
}

// isValidKind 检查 kind name 是否合法（[a-zA-Z0-9_]，非空）。
func isValidKind(s string) bool {
    if len(s) == 0 { return false }
    for _, c := range s {
        if !((c >= 'a' && c <= 'z') || (c >= 'A' && c <= 'Z') ||
            (c >= '0' && c <= '9') || c == '_') {
            return false
        }
    }
    return true
}
```

**向后兼容**：旧格式纯字符串数据（无类型前缀）通过 fallback 自动解码为 `kind:"str"`。

## 六、中间件（可观测性）

```go
// internal/kvspace/middleware.go

// Middleware 是 KVSpace 的包装函数，用于无侵入地叠加 tracing、metrics、logging、retry。
//
// 用法：
//   kv = kvspace.Chain(rawKV,
//       kvspace.WithMetrics(registry),
//       kvspace.WithLogging(logger),
//   )
type Middleware func(KVSpace) KVSpace

// Chain 按顺序叠加多个 Middleware，最后一个在最外层。
func Chain(kv KVSpace, mw ...Middleware) KVSpace {
    for i := len(mw) - 1; i >= 0; i-- {
        kv = mw[i](kv)
    }
    return kv
}
```

Middleware 实现骨架：

```go
type metricsKV struct {
    inner   KVSpace
    counter metrics.Counter
}

func (m *metricsKV) Get(key string) (Value, error) {
    m.counter.Inc("kvspace.get")
    return m.inner.Get(key)
}
// ... 其余方法类似，每个实现 KVSpace 接口即可
```

## 七、builtin 层集成（strconv 消除）

当前 builtin 通过 `string` 往返（`resolveReadValue → string → parseNativeValue`，`writeResult → nativeValue.String() → kv.Set`）。最高标准要求 **builtin 层直接读写 `kvspace.Value`**：

```go
// 最高标准：resolveReadValue 返回 Value（不再是 string）
func resolveReadValue(kv kvspace.KVSpace, framePath, param string) kvspace.Value {
    if len(param) > 0 && param[0] == '"' {
        return kvspace.Str(param[1:]) // 字符串字面量
    }
    if isImmediateNumber(param) {
        if i, err := strconv.ParseInt(param, 10, 64); err == nil { return kvspace.Int(i) }
        if f, err := strconv.ParseFloat(param, 64); err == nil   { return kvspace.Float(f) }
    }
    if isImmediateBool(param) {
        return kvspace.Bool(param == "true")
    }
    var key string
    switch {
    case isRelative(param): key = framePath + "/" + param[2:]
    case isAbsolute(param): key = param
    default:                key = framePath + "/" + param
    }
    v, _ := kv.Get(key)
    return v // 直接返回 Value，保留类型
}

// 最高标准：writeResult 按 nativeValue.kind 写对应类型的 Value
func writeResult(f *op.Frame, result nativeValue) error {
    if len(f.Inst.Writes) > 0 {
        outKey := resolveWriteKey(keytree.FrameRoot(f.PC), f.Inst.Writes[0])
        var v kvspace.Value
        switch result.kind {
        case "int":   v = kvspace.Int(result.i)
        case "float": v = kvspace.Float(result.f)
        case "bool":  v = kvspace.Bool(result.b)
        default:      v = kvspace.Str(result.raw)
        }
        if err := f.KV.Set(outKey, v); err != nil { return err }
    }
    vthread.Set(bg, f.KV, f.Vtid, op.NextPC(f.PC), kvspace.Str("running"))
    return nil
}
```

效果：arithmetic/compare/logic 运算结果以 typed Value 存储，读取时直接 `.Int()`/`.Float()`/`.Bool()`，无 `strconv` 往返。

## 八、vtype 集成：Value 不实现 Dispatch

`vtype` 包导入 `kvspace` 包（`VType.Exec` 参数含 `KVSpace`）。若 `Value` 实现 `Dispatch`，则 `kvspace` 必须导入 `vtype`，形成循环导入：

```
vtype → kvspace → vtype  ✗
```

**`Value` 是纯粹的数据载体，不感知 dispatch**。Dispatch 在 `kvcpu/execute.go` 中完成：

```go
case vtype.Lookup(inst.Opcode) != nil:
    execErr = vtype.Lookup(inst.Opcode).Exec(ctx, c.kv, vtid, pc, inst)
```

`Value.Kind()` 与 `vtype.Name()` 命名对齐是唯一的耦合，无需代码依赖。

## 九、Redis 实现要点

| 方法 | Redis 命令 | 关键点 |
|------|-----------|--------|
| `Get` | `GET` → `.Bytes()` → `DecodeValue` | `.Bytes()` 而非 `.Result()`，避免多余 string 转换 |
| `Set` | `SET key []byte(EncodeValue(v))` | value 以 `[]byte` 传入，go-redis 直接写入 bulk string |
| `GetMany` | `MGET` → `.Result()` → `[]byte(v.(string))` → `DecodeValue` | MGet 无 Bytes()，通过 Go string 中转（binary-safe） |
| `SetMany` | `MSET pairs...` | pairs 中 value 位置传 `[]byte(EncodeValue(v))` |
| `Watch` | `BLPOP key timeout` → `[]byte(vals[1])` → `DecodeValue` | timeout 直接传入 BLPop |
| `Notify` | `LPUSH key []byte(EncodeValue(v))` | value 以 `[]byte` 传入 |
| `checkLink` | `GET` → `.Bytes()` → `raw[0]=='-' && raw[1]=='>'` | 链接不经 EncodeValue，以原始字节判断 |

```go
func (r *redisImpl) Watch(key string, timeout time.Duration) (Value, error) {
    vals, err := r.rdb.BLPop(bg, timeout, resolveCore(key, r.checkLink)).Result()
    if err != nil {
        if errors.Is(err, redis.Nil) { return Value{}, ErrNotFound }
        return Value{}, err
    }
    if len(vals) < 2 { return Value{}, ErrNotFound }
    return DecodeValue([]byte(vals[1])), nil
}
```

## 设计约束汇总

```
kvspace.Value
  ├── 不可变（构造后字段不变）
  ├── owned raw（DecodeValue/Bytes/Raw 构造函数均 copy）
  ├── String() → 调试格式 "kind:repr"，不用于语义解码
  ├── Str()    → 字符串内容，Kind()=="str" 时有效
  └── 无 Dispatch（避免循环导入 vtype）

KVSpace 接口
  ├── 无 context.Context（内存空间抽象，超时由连接层或 Watch timeout 控制）
  ├── Get → Value（不猜类型）
  ├── Watch(key, timeout) → Value（信号语义；超时返回 ErrNotFound）
  ├── Notify(key, Value)（一次性信号，不是持久 Set）
  ├── SetMany 用 []KVPair（有序、类型安全）
  └── DisConn()（断开连接语义）

错误
  ├── ErrNotFound（Get key 不存在 / Watch 超时）
  ├── ErrClosed（连接关闭后操作）
  └── ErrLinkLoop（链接环路）
```
