# ISSUE: 无字符串操作 — 阻塞 ~80 题

kvlang 当前只能按完整字符串读/写，无 charAt, substring, 字符串拼接。

## kvlang 需要
- string[i] 字符索引（对标 Python s[i], C s[i]）
- substring 切片（对标 Python s[i:j]）
- 字符串拼接（+ 已支持，需要扩展到 string kind）
