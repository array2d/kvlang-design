# ISSUE: 无哈希表/字典 — 阻塞 ~120 题

LeetCode 高频需求：Two Sum (O(n)), 3Sum, subarray sum, group anagrams,
first unique char, LRU cache, 等。全部需要 O(1) 查找。

## kvlang 需要
dict/map 类型，对标 Python dict / Go map[string]XValue。
kvspace 路径树天然可以做 hash table (key→path→value)，
需要 {key: val} 字面量语法和 d[key] 读写。
