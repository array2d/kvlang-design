# 001: `/` 是浮点除法，无整数除法

**严重程度**: 高 — 阻塞所有需要整除的算法（LeetCode 中极常见）

**现象**:
```
123 / 10  →  12.3   (float)
123 % 10  →  3      (int, 正确)
```

while 循环 `while (n > 0) { n / 10 -> n }` 永远不会终止，因为 `n` 变成 `0.123`, `0.0123`... 始终 > 0。

**当前 workaround**: 需要两步 + 中间变量:
```
n / 10 -> t
int(t) -> n
```

**建议**: 增加 `//` 整数除法运算符，或让 `/` 在 `int / int` 时自动返回 `int`。
